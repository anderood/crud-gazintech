# crud-gazintechgit


### API
```
GET /dev
```
Retorna todos os devs

```
GET /devs/{id}
```
Retorna o Dev conforme o ID

```
GET /dev/{id}
```
Retorna os dados de um desenvolvedor

```
POST/dev/{id}
```
Adiciona um DEv

```
PUT/dev/{id}
```
Atualiza os dados do Dev

```
DELETE/dev/{id}
```
Delete o Registro
